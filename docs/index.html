Quick Usage

Quick Static Audit

# Basic audit
bun run cli.ts https://example.com

# JSON output
bun run cli.ts https://example.com --format json

# Specific categories
bun run cli.ts https://example.com -c anti-agent-friction,token-economics

# Save to file
bun run cli.ts https://example.com --format md -o report.md

Task Execution with LLM Agent

# Set up LLM key
export LLM_API_KEY=sk-ant-...

# Run a task
bun run cli.ts task https://example.com "Find the product price"

# With stealth mode (bypass bot protection)
bun run cli.ts task https://protected-site.com "Get data" --stealth

# Combined audit + task
bun run cli.ts task https://example.com "Test form" --with-audit --format md

Use as Bun Library

import { runAudit, runTask } from "axagent";

// Static audit
const audit = await runAudit("https://example.com");
console.log(`Score: ${audit.overallScore}/100`);

// Dynamic task
import { createAdapterFromEnv } from "axagent";
const llmAdapter = createAdapterFromEnv();
const task = await runTask("https://example.com", {
  prompt: "Find the price",
  llmAdapter,
  stealth: { enabled: true }
});
console.log(`Result: ${task.agentAnswer}`);

Stealth Mode: Anti-Bot Detection Evasion

Bypass bot protection to audit protected websites

🔍 What It Does

  • Masks navigator.webdriver
  • Spoofs chrome object
  • Rotates User-Agent strings
  • Simulates human-like delays
  • Moves mouse realistically
  • Injects realistic headers
  • Spoof geolocation
  • Masks Function.toString()

⚙️ How to Use

# Enable all stealth features
bun run cli.ts task https://site.com "task" --stealth

# Granular control
--no-stealth-delay    # Disable delays
--no-stealth-mouse    # Disable mouse sim
--no-stealth-geo      # Disable geo-spoofing
--no-stealth-ua       # Disable UA rotation
⚠️ Important: Stealth mode is not guaranteed to bypass all detection. Modern CDNs (Cloudflare, DataDome) may still detect automated traffic. Works best at low frequency with realistic patterns.
